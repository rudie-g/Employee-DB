## Usage ğŸ–¥ï¸ 
---
If you already have a database on MySQL, go quickly through the code for the app and change "employee_db" wherever you find it to whatever the name of your database is. If you do not have a database yet, but have information to input into a database, copy-paste the text in the schema.sql file, located in the db folder, into MySQL workbench and run it. If you want to use premade data to test the app, copy-paste the seed.sql file into MySQL workbench and run it. Once you have your database, run npm start in the command line. The app will take you to the main menu. If this does not work, make sure you have a .env file with the relevant information to access the database in MySQL workbench and try again. ğŸ–¥ï¸
